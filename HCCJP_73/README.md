## 概要

毎月第2金曜日14時からはHCCJPの勉強会！5月は株式会社ネットワールドの後藤さんをお迎えして「Azure Local検証環境」をテーマにお届けします！

## セッション内容

### 1. 専用機がなくても大丈夫！ 既存環境で始めるAzure Local

**後藤 聡 氏**（株式会社ネットワールド 技術本部ソリューションアーキテクト部 エキスパートエンジニア）

プレミアソリューションなどの認定ハードウェアソリューションでの構成が一般的なAzure Localですが、
昨今の半導体供給の問題も相まってPoCや検証などでハードウェアを購入するのはかなりハードルが高いと思います。

実は既存の仮想化環境を利用した「入れ子の仮想化環境（Nested環境）」であれば、意外と簡単に検証環境が
構築できることをご存じでしょうか？

本セッションでは、Hyper-Vをはじめとする仮想化環境下でのNested構成の勘所や、Azure上でのデプロイのコツなど、
簡単に評価環境をデプロイするコツをお伝えします。

### 2. Microsoft "Adaptive Cloud" 最新動向

高添 修 氏（日本マイクロソフト株式会社）

Microsoft高添さんからは毎月恒例のMicrosoft "Adaptive Cloud" の最新動向をお伝えいただきます！
Azure Local、Azure Arc、Windows Server関連の最新情報をお見逃しなく！
